Для использования сервиса необходимо: 1) запустить сервер start_server.py
2) в любом браузере набрать localhost:9000
3) Для действий с шутками необходимо зарегистрировать учетную запить (Имя пользователя минимум 5 символов, пароль минимум 7 символов)


Сервис позволят: 1) Вводить шутку в поле и сохранять нажатием кнопки "Записать шутку"
2) Получить случайную шутку через API (https://geek-jokes.sameerkumar.website/api); записать её в свой список шуток или очистить поле
3) показать все свои записанные шутки; скрыть все записанные шутки
4) ввести ID шутки и нажатием на кнопку "Найти шутку по ID" вывести ее в поле; ввести ID шутки и нажать на кнопку "Удалить шутку по ID" для удаления шутки из базы
5) ввести ID шутки и нажатием на кнопку "Обновить шутку по ID" и ввести тот текст; на который нужно заменить

Для выхода нажать на кнопку "Выход", в левом углу экрана
